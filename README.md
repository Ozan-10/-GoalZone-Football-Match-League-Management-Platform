# ⚽ Premier League Match Management System

Modern yazılım geliştirme prensipleri kullanılarak geliştirilen bu proje, Premier League takımlarını ve maçlarını yönetmek amacıyla oluşturulmuş katmanlı mimariye sahip bir Web API ve Admin Panel uygulamasıdır.

Proje kapsamında kullanıcıların maç, takım ve istatistik verilerini yönetebileceği dinamik bir yönetim paneli geliştirilmiş olup, API tarafında RESTful servis mimarisi uygulanmıştır.

---

# 🚀 Proje Teknolojileri

- ASP.NET Core 8.0
- ASP.NET Core Web API
- ASP.NET MVC
- Entity Framework Core
- MSSQL Server
- N-Tier Architecture
- Repository Design Pattern
- AutoMapper
- LINQ
- Dependency Injection
- Swagger
- RapidAPI
- Bootstrap 5

---

# 📌 Proje Özellikleri

### 🔹 Admin Panel
- Takım ekleme, güncelleme ve silme işlemleri
- Maç oluşturma ve listeleme
- Dashboard istatistik ekranı
- Dinamik veri yönetimi

### 🔹 API Özellikleri
- RESTful API mimarisi
- CRUD operasyonları
- Swagger API dokümantasyonu
- Katmanlı yapı ile sürdürülebilir kod mimarisi

### 🔹 Veritabanı İşlemleri
- Entity Framework Core Code First yaklaşımı
- Migration işlemleri
- MSSQL ilişkisel veritabanı yönetimi
